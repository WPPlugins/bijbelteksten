== Description ==

Bijbeltekst herkent bijbelverwijzingen en verandert deze automatisch in links naar de betreffende bijbeltekst op DeBijbel.nl.

De volgende Bijbelvertalingen worden ondersteund:

* Nieuwe Bijbelvertaling
* Herziene Statenvertaling
* NBG-51 vertaling
* Bijbel in Gewone Taal
* Statenvertaling editie 1977
* Groot Nieuws Bijbel
* Willibrordvertaling editie 2012
* Biebel in t Grunnegers
* Nije Fryske Bibeloersetting

Daarnaast is het mogelijk om de NBV-studieaantekeningen te gebruiken.


De plugin kan op twee manieren werken:

* Dynamisch - links worden gemaakt bij het laden van de pagina (standaard)
* Statisch - links worden ‘ingebakken’ in de tekst

Bijbelteksten wordt u gratis aangeboden door de Bijbelse Cultuur Stichting.

== Installation ==

Het installeren van deze plugin is heel eenvoudig.

1. Download de plugin en pak het bestand bijbelteksten.zip uit.
2. Upload de map ‘bijbelteksten’ naar de Wordpress plugins map, meestal is dit http://jouwwebsite/wp-content/plugins
3. Activeer de plugin. Dit kan via Plugins > Geïnstalleerde plugins. 
4. In het menu Instelling is nu een nieuwe optie verschenen > Bijbelteksten.
5. In principe hoef je niets aan de instellingen te veranderen. 

== Frequently Asked Questions ==

= Worden alle bijbelverwijzingen herkend? =

Nee, niet alle notaties worden herkend. 
- Tweeletterige afkortingen van bijbelboeken veroorzaken vaak moeilijkheden. Wil je verwijzen naar Johannes 3 vers 16, schrijf dan niet Jh 3:16, maar Joh 3:16. Overigens maakt het niet uit of je wel of geen punt zet achter de afkorting van het bijbelboek.
- Als je wilt verwijzen naar bijbelboeken waar meerdere delen van zijn (bijv. Koningen of Samuel) dan moet er een spatie volgen na het deel, dus bijvoorbeeld: 1 Koningen. Een verwijzing naar 1Koningen zal namelijk niet worden herkend.

= Krijgen bezoekers geen inlogscherm meer te zien op debijbel.nl? =

Dat ligt er aan. Als de bezoeker standaard is ingelogd, dan niet. Anders krijgt de bezoeker een optie om zich in te loggen of (gratis) te registreren. Na het inloggen verschijnt de opgevraagde bijbeltekst. Schrik niet, de bezoeker hoeft écht niet elke keer in te loggen als hij debijbel.nl gebruikt. Meestal is dit maar eenmalig, dus het probleem lijkt groter dan het is. 

= Kan de plugin ook verwijzen naar Engelse Bijbels? =

Nee, deze plugin is uitsluitend bedoeld om te verwijzen naar Nederlandse Bijbelvertalingen. Wil je verwijzen naar anderstalige Bijbels, dan raden wij je aan gebruik te maken van een plugin als Scripturizer. 

= Kan ik deze plugin gebruiken in combinatie met een plugin als Scripturizer? =

Liever niet, dit zal bijna altijd fouten opleveren. Bovendien komt het de laadtijd van de website niet ten goede.

= Is deze plugin volledig gratis? =

Ja, je kunt alle functionaliteit gratis gebruiken. Maar mocht je deze plugin waarderen, overweeg dan eens een gift over te maken naar de Bijbelse Cultuur Stichting (http://www.bijbelsecultuur.nl).
